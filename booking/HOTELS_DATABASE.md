## 🔗 How Hotels Work in the System

### In Bookings
When a user books a tour:
1. Select optional hotel addition
2. Hotel is linked to booking
3. Hotel cost calculated: `Price/Night × Number of Days`
4. Added to total booking price

### Total Price Calculation
```
Total = (Tour Base Price × Days × Discount) 
        + (Hotel Price × Days) 
        + Sum(Add-on Prices)
```

### Example Booking
**Tour:** Pokhara 3 Days
- Base Price: Rs. 8,000
- Days: 3
- Hotel: Fish Tail Lodge (Rs. 9,000/night)
- Addons: None

**Calculation:**
- Tour Cost: 8,000 × 3 × 0.85 (discount) = Rs. 20,400
- Hotel Cost: 9,000 × 3 = Rs. 27,000
- **Total: Rs. 47,400**

## 🛠️ Database Operations

### View All Hotels (Admin Panel)
- Go to http://127.0.0.1:8001/admin/
- Click "Hotels"
- See all 36 hotels with their prices

### Add New Hotel
```bash
# Via admin interface
1. Login to admin
2. Click "Hotels" → "Add Hotel"
3. Enter name and price per day

# Via Django shell
python manage.py shell
>>> from tour.models import Hotel
>>> Hotel.objects.create(name="Hotel Name", price_per_day=5000)
```

### Update Hotel Price
```bash
python manage.py shell
>>> from tour.models import Hotel
>>> hotel = Hotel.objects.get(name="Hotel Name")
>>> hotel.price_per_day = 6000
>>> hotel.save()
```

### Delete Hotel
```bash
# Via admin or shell
>>> hotel.delete()
```
